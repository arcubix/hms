# IPD Reports Access Guide

## How to Access IPD Reports with Sample Data

### Step-by-Step Navigation Path:

1. **Start from the Main Dashboard**
   - When you load the application, you'll see the IPD Management System

2. **Open the Left Sidebar Navigation**
   - Look for the dark sidebar on the left side of the screen
   - It contains various IPD module options

3. **Click on "IPD Reports"**
   - Scroll down in the sidebar menu
   - Find and click the button labeled "IPD Reports" (has a FileText icon)
   - This will switch the main content area to show the Reports Listing page

4. **Browse Report Categories**
   - You'll see 9 main report categories displayed as cards:
     1. **Admission & Discharge Reports** (Blue)
     2. **Billing & Finance Reports** (Green)
     3. **Clinical & Medical Reports** (Purple)
     4. **Bed & Room Management Reports** (Orange)
     5. **Pharmacy & Consumable Reports** (Teal)
     6. **Laboratory & Radiology Reports** (Light Blue)
     7. **OT (Operation Theatre) Reports** (Red)
     8. **Panel / Insurance Reports** (Yellow)
     9. **Compliance & Audit Reports** (Gray)

5. **Select a Specific Report**
   - Click on any report card (e.g., "Daily Admissions Report")
   - This will open the detailed report view

6. **View Sample Data**
   - The report will display with:
     - Report title and description
     - Filter options (Date Range, Ward, Consultant, etc.)
     - Data table with realistic Indian patient names
     - Summary statistics
     - Export options (PDF, Excel, Print)
     - Charts and visualizations where applicable
